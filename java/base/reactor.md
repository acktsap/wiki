## 5. Kotlin Support

### Null Safety

- java는 null safefy하지 않지만 reactor에서 자체적으로 null safety를 처리하기도 하고 (flux에서 null 안되는거 같음) `reactor.util.annotation`에 있는 annotation들로 null 처리도 해주고 있음.

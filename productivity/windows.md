## Shortcut

virtual desktop

- start : Ctrl + Window + D
- move : Ctrl + Window + <- or ->
- remove : Ctrl + Window + F4

etc

- cortana start : Window + s or touch touchpad with three fingers
- explorer : Window + E
- setting : Window + I
- for super user : Window + X
- lock : Ctrl + L

execution

- 방화벽 : firewall.cpl
- 메모장 : notepad
- 그림판 : mspaint
- 캡쳐 도구 : snippingtool
- 설정? : msconfig
- 레지스트리 : regedit
- 시작프로그램 : shell:startup

visual studio shortcuts

- auto formatting : ctrl + k + f

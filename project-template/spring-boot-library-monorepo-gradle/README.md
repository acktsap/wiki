## Build from source

### Prerequisites

- Jdk8 or higher

### Build

- Clean: `./gradlew clean`
- Check: `./gradlew check --parallel`
  - Coverage report : `${buildDir}/kover/html/index.html`
- Assemble: `./gradlew build --parallel`
- Install to local: `./gradlew install`
- Publish: `./gradlew publish`

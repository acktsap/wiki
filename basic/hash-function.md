## What?

- 임의의 size의 데이터를 fixed-size value로 바꾸는 함수. 바뀐 값은 hash value, digests, hashes, hash code라고 부름.
- 데이터를 nearly constant time에 조회하기 위해 사용.

## Good hash function?

- fast to compute.
- minimum collision.

## & Hash Table

- hash table에서 hash code 값을 계산할 때 hash function이 사용됨.
